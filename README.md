# Python Snippets

A reusable toolkit of generic Python scripts for data analysis,
automation, visualization, APIs, and SQL.

## Folder structure

- api/  
  Generic API helpers and web scraping utilities (BeautifulSoup)  
  - Extract links  
  - Extract HTML tables  

- data-cleaning/  
  Text and data cleaning utilities  

- visualization/  
  Reusable plotting functions  

- excel-automation/  
  Excel file creation helpers  

- sql/  
  SQL query templates  

- pandas/  
  Common DataFrame operations  
  Dataframe explortation helpers
  Basic statistics utilities

## Philosophy

- One file = one reusable idea  
- No course-specific code  
- Designed for reuse and long-term memory aid
