# Python Snippets

A reusable toolkit of generic Python scripts for data analysis,
automation, visualization, APIs, and SQL.

## Folder structure

- api/              Generic API helpers
- data-cleaning/    Text and data cleaning utilities
- visualization/    Reusable plotting functions
- excel-automation/ Excel file creation helpers
- sql/              SQL query templates
- pandas/           Common DataFrame operations

## Philosophy

- One file = one reusable idea
- No course-specific code
- Designed for reuse and memory aid
